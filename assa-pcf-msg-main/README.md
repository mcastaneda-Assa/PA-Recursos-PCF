# pcf-msg

Control responsivo para Power Apps que renderiza archivos .msg en formato base64.

## Requisitos

- Node.js 14+
- npm 6+
- Power Apps CLI

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm start
npm start:watch
```

## Build

```bash
npm run build
```

## Lint

```bash
npm run lint
npm run lint:fix
```

## Limpieza

```bash
npm run clean
```
